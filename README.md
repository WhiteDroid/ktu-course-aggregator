# ⚡ KTU Insight Engine 2.0

A Spatial Data Experience and AI-powered Sentiment Dashboard built for Kerala Technological University (KTU) students.

### 🌟 Features
* **Bento Grid UI:** A sleek, modular, glassmorphic design with 3D parallax hover physics.
* **Auto-Rotating Personas:** 8 dynamic Light and Dark mode themes that adapt typography and color gradients automatically.
* **NLP Sentiment Analysis:** Uses `TextBlob` to instantly calculate and score college and course reputation.
* **RAG AI Chat:** "Chat with reviews" using a keyword-based synthesis engine to get direct answers from student feedback.
* **Geospatial Heatmap:** A live map built with Plotly Mapbox plotting the sentiment of KTU colleges across Kerala.
* **Enterprise Anti-Spam:** Rate limiting, regex filtering, and automated avatar generation.
